Make adjustments without knowing how to code, send it to a developer. Simple and easy.

== Description ==

The world is a ever changing place, and so is wordpress!
Maybe you have come upon a problem you cannot solve by your own hands. Well, maybe you can solve it, but who has the time?
With Adjustme, you get help with a few clicks.

This is how it works.
*After installing the plugin, you visit the page where you have something you want to fix, change, or adjust.
*In the toolbar, click the Adjustme button.
*Point to where you want the change, click, and write about it.
*Got more adjustment on the same page? Just add more pins.
*When you are done, simply "Send to developer" and you can sit back and relax while we take a look at it. We will come back to you with a changed version, when you approve the changes we will upload them to your site.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Now you see a box where you can enter your email, do that.
1. Use the Settings->Adjustme screen to configure the plugin
1. Go to the front-end and click the Adjustme icon in the upper left corner.
